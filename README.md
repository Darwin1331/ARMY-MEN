# ARMY MEN - Catálogo Web

Catálogo web desarrollado para la tienda **ARMY MEN**, especializado en la exhibición de equipamiento táctico y militar.

## Descripción

Este proyecto consiste en un catálogo web moderno y responsive que permite a los clientes explorar los productos disponibles de forma rápida e intuitiva.

El sitio está diseñado para funcionar como un catálogo informativo, sin carrito de compras ni sistema de pagos.

## Características

- Buscador de productos.
- Filtro por categorías.
- Ordenamiento A-Z y Z-A.
- Paginación automática.
- Modal con información detallada del producto.
- Ampliación de imágenes.
- Diseño responsive.
- Integración con Google Maps.
- Contacto directo mediante WhatsApp.
- Carrusel con fotografías de la tienda.
